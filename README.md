# Student Management System

## Overview

A role-based student management system developed using PHP and MySQL. The project allows teachers and students to access different functionalities through separate login modules and database-driven operations.

## Features

* Teacher and student login system
* Add student functionality
* Notes management system
* CRUD operations for student records
* Course management
* Database integration using MySQL
* Responsive frontend interface
* Hosted online using InfinityFree

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
